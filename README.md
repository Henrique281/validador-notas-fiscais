# Validador de Notas Fiscais

Projeto em Python para análise de notas fiscais XML.

## Funcionalidades

- leitura de XML de NF-e
- detecção de notas duplicadas
- detecção de CNPJ suspeito
- detecção de valores fora da média
- estatísticas de faturamento
- geração de relatório CSV

## Tecnologias

- Python
- XML
- CSV

## Como executar

Clone o repositório:

git clone https://github.com/SEU_USUARIO/validador-notas-fiscais.git

Entre na pasta:

cd validador-notas-fiscais

Execute:

python main.py
